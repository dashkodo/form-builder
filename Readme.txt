База з питаннями в файлі questions.txt
Формат файлу


[назва секціі]
QuestionType=<тип питання, підтримуються Longtext, Text, Radio, Checkbox, Date. Moжна додавати required, наприклад Date required>
Текст питання
Поясненн для питання якщо треба, необовязково
    Варіант1
    Варіант2[free]
---
QuestionType=Radio required
Згода на обробку персональних даних
Я підтверджую, що ознайомлений(а) з метою збору та обробки моїх персональних даних і надаю добровільну згоду на їх обробку, зберігання та використання відповідно до вимог Закону України «Про захист персональних даних». Надані мною дані можуть використовуватися виключно з метою проведення опитування, тестування, оцінювання або виконання інших завдань, визначених організатором збору інформації, та не підлягають передачі третім особам без законних підстав.
    Я підтверджую
---
Для текстових питаннь і дати, варіанти відповіді не працюють.
модифікатор [free] для варіанта відповіді додасть необовязкове текстове поле для даного варіанта.

Результат зберігається у data.csv

для запуску ->
1) Створити .instatunnel.env файлу
2) Запустити Start.bat
